# Olist E-Commerce Sales Data Analysis

## Overview
This project is an end-to-end **E-Commerce Sales Data Analysis** using the Brazilian Olist dataset.  
The goal of this project is to clean raw data, perform exploratory data analysis (EDA), visualize trends, and extract meaningful business insights.

This project demonstrates how real-world data can be transformed into actionable insights for decision-making.


## Objectives
- Understand customer purchasing behavior
- Analyze sales trends over time
- Explore order status patterns
- Evaluate delivery performance
- Identify top-performing product categories
- Generate business-focused insights

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook


## Dataset
Source: Brazilian Olist E-Commerce Dataset  

The dataset contains:
- Customers
- Orders
- Products
- Sellers
- Payments
- Reviews
- Geolocation

## Key Analysis Performed
- Data cleaning & preprocessing
- Handling missing values
- Feature exploration
- Exploratory Data Analysis (EDA)
- Sales trend analysis
- Order status distribution
- Delivery time performance
- Data visualization

## Visualizations
This project includes:
- Bar charts
- Line charts
- Distribution plots
- Correlation heatmaps
- Category-wise performance charts
  

  ## Business Insights ##
- dentified high-revenue product categories
- Analyzed delivery delays and their impact on customer satisfaction
- Studied preferred payment methods
- Explored repeat customer behavior
- Highlighted key sales trends and seasonal patterns
